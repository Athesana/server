JSP(Java Server Page)

HTML 형식에서 자바 코드를 사용할 수 있는 "자바 언어(스크립트 언어)"이다.
서블릿에서 Java 코드로 응답 HTML을 구현했던 복잡함을 간단하게 작성, 해결할 수 있다.

1. JSP 스크립트 요소(Elements)

JSP 페이지에 자바 코드를 직접 기술할 수 있게 해주는 기능이다.

  1. 선언문 : <%! 자바코드 %> = 필드(멤버 변수)와 메소드를 선언하기 위해서 사용한다.
  2. 스크립트릿 : <% 자바코드 %> = JSP에서 자바 코드를 사용하기 위해서 사용한다. 함수 안에서 자바 코드를 사용하는 부분이라서 ; 사용한다. for 클라이언트에게 출력하기 위해서 out.write() 직접 사용할 수도 있다.
  3. 표현식 : <%= 자바코드 %> = 자바에서 작성한 값을 출력하기 위해서 사용한다.
